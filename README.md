**Project**
End-to-End Sales & Finance Analytics Dashboard using Advanced Excel | AtliQ Hardwares

This project delivers an end-to-end analysis of sales and financial performance of AtliQ Hardwares using Advanced Excel.
It focuses on extracting actionable insights related to revenue growth, customer contribution, market performance, and profitability trends. The dashboard enables stakeholders to monitor KPIs and make data-driven business decisions effectively.

**Process & Steps**
To build this report, I used several tools and techniques:
ETL :- Extracted, transformed, and Load
Power Query :- For efficent data transformation
Data Modelling :- Created a clear model to connect sales and finance data
DAX Measures :- Added key metrics and Calculations
Pivot Tables :- For easy data exploration and insights
Conditional Formatting :- Highlighted important trends and outliers

**Features & Highlights**

📊 Analyzed revenue growth ($87.5M → $598.9M, +304%)

🏆 Identified top customers driving revenue (Amazon, AtliQ e Store)

🌍 Evaluated market performance vs targets (identified gaps across regions)

💰 Performed P&L analysis (declining GM% from 41.4% → 36.4%)

📅 Discovered monthly & seasonal sales trends

🎛️ Built dynamic dashboard with slicers for easy insights
